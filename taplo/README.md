# taplo plugin

[taplo](https://github.com/tamasfe/taplo) plugin for [proto](https://github.com/moonrepo/proto).

## Installation

taplo is not built into proto, so register this plugin, pin a version, then install:

```shell
proto plugin add taplo "https://raw.githubusercontent.com/Genesis-Embodied-AI/proto-plugins/main/taplo/plugin.toml"
proto pin taplo latest --resolve
proto install taplo
```
