# prek plugin

[prek](https://github.com/j178/prek) plugin for [proto](https://github.com/moonrepo/proto).

## Installation

prek is not built into proto, so register this plugin, pin a version, then install:

```shell
proto plugin add prek "https://raw.githubusercontent.com/Genesis-Embodied-AI/proto-plugins/main/prek/plugin.toml"
proto pin prek latest --resolve
proto install prek
```
