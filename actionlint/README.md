# actionlint plugin

[actionlint](https://github.com/rhysd/actionlint) plugin for [proto](https://github.com/moonrepo/proto).

## Installation

actionlint is not built into proto, so register this plugin, pin a version, then install:

```shell
proto plugin add actionlint "https://raw.githubusercontent.com/Genesis-Embodied-AI/proto-plugins/main/actionlint/plugin.toml"
proto pin actionlint latest --resolve
proto install actionlint
```
