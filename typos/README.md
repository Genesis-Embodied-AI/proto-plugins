# typos plugin

[typos](https://github.com/crate-ci/typos) plugin for [proto](https://github.com/moonrepo/proto).

## Installation

typos is not built into proto, so register this plugin, pin a version, then install:

```shell
proto plugin add typos "https://raw.githubusercontent.com/Genesis-Embodied-AI/proto-plugins/main/typos/plugin.toml"
proto pin typos latest --resolve
proto install typos
```
