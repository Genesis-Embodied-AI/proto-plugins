# dprint plugin

[dprint](https://dprint.dev) plugin for [proto](https://github.com/moonrepo/proto).

## Installation

dprint is not built into proto, so register this plugin, pin a version, then install:

```shell
proto plugin add dprint "https://raw.githubusercontent.com/Genesis-Embodied-AI/proto-plugins/main/dprint/plugin.toml"
proto pin dprint latest --resolve
proto install dprint
```
