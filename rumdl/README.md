# rumdl plugin

[rumdl](https://github.com/rvben/rumdl) plugin for [proto](https://github.com/moonrepo/proto).

## Installation

rumdl is not built into proto, so register this plugin, pin a version, then install:

```shell
proto plugin add rumdl "https://raw.githubusercontent.com/Genesis-Embodied-AI/proto-plugins/main/rumdl/plugin.toml"
proto pin rumdl latest --resolve
proto install rumdl
```
